# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-03-21

### Added

- Initial release
- Popup Button block with trigger (button/link) and overlay modal
- InnerBlocks with Cover block for popup content
- Custom colours (text, background, hover)
- Popup sizes, padding, border radius options
- Accessibility: ARIA attributes, focus trap, Escape to close, `prefers-reduced-motion`
- Deprecated block versions for backwards compatibility
- Documentation: README, ARCHITECTURE, CONTRIBUTING
- Security: script versioning, server-side attribute documentation
